```
│                                                                             │
│  예상 TAM (Total Addressable Market): 약 50만개 기업                        │
│  예상 SAM (Serviceable Available Market): 약 15만개 기업                    │
│  초기 목표 SOM: 1,000개 기업 (3년 내)                                       │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 1.2 Value Proposition

| 구분 | 기존 ERP (on-premise) | K-ERP SaaS |
|------|----------------------|------------|
| 초기 비용 | 수천만원~수억원 | 무료~수백만원 |
| 월 비용 | 유지보수비 (고정) | 사용량 기반 |
| 도입 기간 | 3~12개월 | 즉시~1주 |
| 업데이트 | 별도 비용/작업 | 자동 (무료) |
| 정부 연동 | 수동/별도 연동 | 자동 연동 |
| 확장성 | 추가 비용 | 자동 확장 |

### 1.3 핵심 차별화 요소

1. **무료 우선 전략**: 홈택스 스크래핑, NTS API 우선 사용으로 비용 절감
2. **자동화**: 세금계산서 자동 수집, 4대보험 자동 신고
3. **한국 특화**: K-IFRS 완벽 지원, 한국 정부기관 완벽 연동
4. **클라우드 네이티브**: 언제 어디서나 접속, 자동 백업

### 5.3 업셀/크로스셀 전략

```

- 사용량 80% 도달 시 알림
- Standard 기능 미리보기 제공
- 업그레이드 시 1개월 무료

Standard → Enterprise 전환 유도:
- 다중 회사 필요 시
- API 연동 요청 시
- 보안/컴플라이언스 요구 시
```
