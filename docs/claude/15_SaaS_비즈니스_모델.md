## 3. SaaS 가격 모델 분석

### 3.1 가격 모델 유형별 비교

| 모델 | 설명 | 장점 | 단점 | K-ERP 적합도 |
|------|------|------|------|-------------|
| **정액제 (Flat-rate)** | 단일 가격 | 단순, 예측가능 | 소규모 손해 | ⭐⭐⭐⭐ |
| **계층형 (Tiered)** | 기능별 플랜 | 업셀 용이 | 복잡 | ⭐⭐⭐⭐⭐ |
| **사용자 기반 (Per-user)** | 인원당 과금 | 확장 수익 | 도입 저해 | ⭐⭐⭐ |
| **사용량 기반 (Usage-based)** | 실제 사용량 | 공정함 | 예측 어려움 | ⭐⭐ |
| **프리미엄 (Freemium)** | 무료+유료 | 확산 용이 | 전환 어려움 | ⭐⭐⭐ |
| **기능 기반 (Feature-based)** | 모듈별 과금 | 맞춤화 | 복잡 | ⭐⭐⭐ |

### 3.2 권장 모델

**계층형 (Tiered) + 정액제 (Flat-rate) 하이브리드**

- 각 Tier 내에서는 정액제로 단순화
- Tier 간 명확한 기능 차별화로 업셀 유도
- 추가 사용자 과금으로 확장 수익 확보

## 5. 핵심 SaaS 지표

### 5.1 벤치마크 기준 (B2B SaaS)

| 지표 | 업계 평균 | K-ERP 목표 | 비고 |
|------|----------|-----------|------|
| **월간 이탈률 (Churn)** | 3.5% | **< 2.5%** | 연간 ~30% 유지 |
| **연간 이탈률** | 4.9% | **< 4%** | 고객 유지 중요 |
| **LTV:CAC 비율** | 3:1 | **4:1** | 건강한 수익성 |
| **CAC 회수 기간** | 12개월 | **< 10개월** | 빠른 ROI |
| **NRR (순수익유지율)** | 100~110% | **> 105%** | 업셀 포함 |
| **ARPU (사용자당 수익)** | - | **6.5만원** | 가중 평균 |

### 5.2 지표 계산 공식

```
CAC (고객획득비용) = 마케팅+영업 비용 / 신규 고객 수

LTV (고객생애가치) = ARPU × 평균 고객 유지 기간
                  = ARPU / 월간 이탈률

LTV:CAC 비율 = LTV / CAC
             ≥ 3:1 (건강한 수준)
             ≥ 4:1 (우수한 수준)

NRR (순수익유지율) = (시작 MRR + 확장 - 축소 - 이탈) / 시작 MRR × 100%
```

### 5.3 목표 지표 달성 전략

| 지표 | 목표 | 달성 전략 |
|------|------|----------|
| 이탈률 < 2.5% | 고객 성공 프로그램, 조기 경보 시스템 |
| LTV:CAC > 4:1 | 파트너 채널 확대, 콘텐츠 마케팅 |
| CAC 회수 < 10개월 | 정부 바우처 활용, 연간 계약 유도 |
| NRR > 105% | Starter→Business 업셀, 추가 사용자 |

## 8. 고객 유지 전략
